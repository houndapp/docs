§Sans-IO network impairment simulator

A deterministic, virtual-clock model of a lossy network link, used to test the media engine without real sockets. You pump it: send(now_us, packet) to enqueue, poll(now_us) to collect whatever has arrived by now_us. It models, all reproducibly from a seed:

  • loss — per-packet drop probability;
  • duplication — per-packet duplicate probability;
  • propagation + jitter — base one-way delay plus uniform random jitter (which naturally produces reordering);
  • bandwidth + queue — a serialization model where sending faster than the link capacity builds queuing delay, and a bounded queue drops on overflow (congestion loss).

This is the foundation for the reliability, congestion-control, and jitter-buffer tests. It is intentionally pure and dependency-free.
