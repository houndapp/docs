Module layer 

Source
Expand description

The video layer model — the single source of truth mapping a layer combo to a moq track name + wire TrackKind, and the pure helpers that turn a receiver’s tile size / bandwidth into a QualityRequest.

Two scalability axes, deliberately mapped to two different transport mechanisms:

  • Spatial (SpatialLayer) = a SEPARATE moq track. The relay is forward-no- transcode + group-atomic, so per-receiver selection is per-track subscription; the source layer keeps its original track name (an un-upgraded peer never learns about layers and keeps working), and the low layer is an additional track a constrained / small-tile receiver subscribes to instead.
  • Temporal (TemporalLayer) = a per-FRAME marking WITHIN one spatial track (a base frame is self-decodable; an enhancement frame depends on base and is droppable). Not part of the track identity — see the producer-side gating.

Keeping this here (not in call.rs) keeps the wire/string juggling in one tested place and out of the IO glue.

Constants§

LOW_LAYER_TARGET_BPS
The bitrate a tile-driven low request caps to. The producer coalesces every receiver’s cap into ONE shared low layer sized to the most-constrained requester, so this is the target the low encoder runs at when small tiles ask for it.
TILE_SOURCE_MIN_PIXELS
A rendered tile smaller (in pixels) than this wants the low layer, not the source. ~360p — below it the source’s extra resolution is wasted on a thumbnail.

Functions§

most_constrained
Fold two demands into the more-constrained one (Capped beats Source; the lower cap wins). Used to combine a tile-size demand with a bandwidth demand so the receiver can never over-request beyond what either signal allows.
tile_quality_request
Map a rendered tile’s pixel size + visibility to a QualityRequest. A hidden or small tile wants the low layer; a large, visible tile wants the source. This is the demand half of demand-driven simulcast — it never asks for more than the tile can use, which is exactly the receiver-CPU win (a thumbnail pulls the low track).
track_kind
The wire TrackKind for a (source, spatial) pair.
track_name
The moq track_name for a (source, spatial) pair — the subscribe/publish key under the per-peer broadcast path. The source layers keep the original names so a peer that never learns about layers is unaffected.
