Expand description

The fixed-size media packet header. See the module docs for the byte layout.

Fields§

§version: u8§codec: Codec§track: TrackKind§sender: u16§packet_seq: u32

Per-(sender, track) packet sequence number (wraps at 2^32).

§frame_seq: u32

Per-(sender, track) frame number; shared by all fragments of a frame.

§frag_index: u16

0-based fragment index within the frame.

§frag_count: u16

Total fragments for this frame (always ≥ 1).

§timestamp_us: i64

Capture timestamp in microseconds on the sender’s shared clock.

§keyframe: bool

First frame of a decodable sequence (video keyframe; ignored for audio).

§end_of_frame: bool

Last fragment of the frame (RTP “marker” bit).

Source

pub fn parse(buf: &[u8]) -> Result<Self, PacketError>

Parse a header from the first 28 bytes of buf.

Returns an error if the buffer is too short, the version is unknown, or a codec/track byte is unrecognised. Does not validate frag_* relationships (that is the reassembler’s job, which must tolerate hostile input).
