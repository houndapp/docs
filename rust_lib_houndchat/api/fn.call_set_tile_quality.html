Expand description

Report a remote video tile’s rendered pixel size + visibility, for peer peer_id’s source (Source::Camera | Source::Screen). Small or hidden tiles are switched to the LOW simulcast layer (and large/visible ones to the full source), so a receiver only pays to decode what it’s actually showing. Call from each tile’s layout/visibility. No-op when not in a call or for a non-video source.
