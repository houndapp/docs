§SFU client

Manages the iroh QUIC connection from the Tauri app to the SFU service.

§Lifecycle

  1. SfuClient::connect() — binds an iroh endpoint, dials the SFU, performs the Hello/HelloAck handshake, and starts a background task that reads incoming frames and emits Tauri events.
  2. SfuClient::send_frame() — encodes and fragments a frame, then sends it via QUIC datagrams (delta frames) or a unidirectional stream (keyframes).
  3. Dropping SfuClient closes the connection cleanly.

§Frame wire format

Each wire packet is FrameHeader::BYTES || payload_fragment. The SFU knows our peer ID (from the HelloAck) so it can route incoming frames. The Tauri event listener on the TypeScript side calls deliverFrame(buf) which passes the raw bytes to the MediaWorker for header parsing.
