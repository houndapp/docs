§Voice subsystem

This module owns all of the Rust-side voice channel functionality:

  • commands — Tauri commands called by the TypeScript frontend (voice_join, voice_leave, voice_list_capture_targets).

  • state — VoiceState held in Tauri managed state for the lifetime of the app. Tracks whether there’s an active voice session and stores the relay token.

  • screen — Platform-agnostic screen capture. The PlatformCapturer alias resolves to the right implementation for the current OS without any conditional logic in callers.

  • error — VoiceError enum, serialisable as {kind, message} for clean TypeScript error handling.
