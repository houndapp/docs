pub(super) async fn establish_live(
    params: &ConnectionParams,
) -> Result<LiveSession>
Expand description

Build one full control+MoQ LiveSession. Each establish re-mints the relay token by calling voice/join, so a token-expiry SFU drop heals via reconnect — that re-mint is what keeps a call alive past the relay token’s TTL. The capture loops subscribe to params.devices.* (so device selection survives a reconnect) and read params.mute/params.camera_enabled. Seeds the identity map from the fresh voice/join each time.
