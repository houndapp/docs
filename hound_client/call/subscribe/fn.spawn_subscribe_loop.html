Expand description

Spawn the subscribe loop: watch the SessionView, and for every remote track start a per-track pipeline (subscribe → decode → sink for video, mix bus for audio); stop pipelines for tracks that vanish. Sends Subscribe/keyframe intent on the control plane as tracks appear.
