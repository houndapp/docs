pub(super) async fn dial_relay(
    relay_url: &str,
    sfu_node_id: &str,
) -> Result<(Endpoint, Connection)>
Expand description

Dial the MoQ relay (its node id + relay URL) on a fresh iroh endpoint, returning the QUIC connection the MoQ session rides. The endpoint is per-call with an ephemeral key (no persisted device identity yet) — fine for the call’s lifetime; sharing it across calls is a later optimization. Returns BOTH the endpoint and the connection: the caller MUST keep the [Endpoint] alive for the connection’s lifetime. The SFU is reachable only via the relay (it has no client-routable direct address), so all packets — including every stream after the handshake — ride the endpoint’s magicsock/relay actor. Dropping the endpoint stops that actor, so the QUIC handshake still completes but the very next stream silently never transmits (the relay’s accept_bi then times out). Holding only the Connection is not enough.
