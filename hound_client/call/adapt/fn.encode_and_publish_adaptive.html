The shared per-frame encode→adapt→publish core for both video capture loops. Retargets the encoder to the BWE’s latest bitrate (only on a real move), honours a throttle-approved keyframe (swap-clears so it fires once), downsamples to the rung’s fps, downscales to the rung geometry when smaller than capture, encodes, and forwards each access unit via forward. forward returns false on backpressure (its bounded channel was full); the caller’s drop bumps dropped, which the adaptation tick reads as the congestion signal. A forced keyframe is re-armed when it is dropped, so the next captured frame re-emits it.
