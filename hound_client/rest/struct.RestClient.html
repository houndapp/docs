Struct RestClient 

Source 
pub struct RestClient {
    http: OnceLock<Client>,
    base: Url,
    token: ArcSwapOption<String>,
}
Expand description

HTTP client bound to one backend base URL, carrying the optional session token. Cheap to clone-share via Arc<RestClient>; the token cell is interior-mutable.

Fields§

§http: OnceLock<Client>

Built on first request, not at construction: reqwest::Client::new runs macOS system-proxy discovery (SCDynamicStore), which stalls ~2 s in unbundled binaries (dev runs, every per-test process). Deferring it keeps constructors instant and moves the one-time cost onto the first actual HTTP call, off the caller’s startup path.

§base: Url§token: ArcSwapOption<String>

Current bearer token (None when signed out). Swapped atomically by auth.
