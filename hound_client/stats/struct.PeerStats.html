Struct PeerStats 

Source 
pub struct PeerStats {
    audio: InboundAudioStats,
    screen_audio: InboundAudioStats,
    video: InboundVideoStats,
    screen_video: InboundVideoStats,
}
Expand description

One remote peer’s inbound pipeline stats (audio + video). Created lazily when the peer’s first pipeline starts; removed when both pipelines tear down.

Fields§

§audio: InboundAudioStats

The peer’s microphone (mic track).

§screen_audio: InboundAudioStats

The peer’s shared screen-share app audio (screen_audio track), counted separately from the mic so the panel shows whether screen audio is flowing.

§video: InboundVideoStats

The peer’s camera (cam track) inbound video.

§screen_video: InboundVideoStats

The peer’s screen-share (screen track) inbound video, counted separately from the camera so a peer sharing BOTH does not have the two streams’ frames summed into one decoded_fps. See InboundVideoKind.
