Quiet cargo builds, separated from runtime logs. Every binary the harness runs (backend, houndbot, snapdiff, the nextest test binaries) is built HERE first — full compiler output captured into the run’s build.log, surfaced only on failure — and then executed directly, so each runtime artifact (backend.log, bot logs, test output) contains runtime lines only, never pages of “Compiling…” chatter.
