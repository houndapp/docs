Wire types for the call control plane.

A structured, versioned message protocol carried over WebSocket. It manages the media session — tracks, codecs, mute, keyframe requests, reconnect/resync — on top of the existing participant roster (HTTP join + socket.io). It is transport-agnostic: these types serialize with serde (JSON on the wire) and carry no I/O.

Which video codecs a peer can decode, advertised at join. The sender of a video track intersects this across every other participant and encodes the highest-priority codec they can all decode (the forward-no-transcode SFU sends one encoding to all subscribers, so the sender must satisfy the weakest peer).
