pub fn join( &mut self, client_version: u16, decode_caps: DecodeCaps, ) -> Result<(PeerId, Vec<Outgoing>), ServerMessage>

Admit a peer that sent ClientMessage::Hello (carrying its decode_caps). On a compatible version, assigns a PeerId and returns it with the outgoing messages: a Welcome snapshot to the joiner (the full roster, each peer’s decode caps included, so the joiner can immediately pick a codec everyone plays) and a PeerJoined carrying the joiner’s caps to everyone else (so existing producers can re-negotiate down if needed). On a version mismatch, returns Err with the Error to send before closing.

pub fn leave(&mut self, peer: PeerId) -> Vec<Outgoing>

Remove a peer (graceful Bye or a dropped connection). Drops its tracks and subscriptions and announces PeerLeft to the rest. Idempotent: a peer that isn’t present yields no messages.
