Expand description

Begin a browser-deferred login — the passkey path on platforms with no system passkey provider (Linux), same flow as houndctl auth login. Binds the loopback callback, opens the system browser at the hosted auth page (auth_url, e.g. https://auth.hound.chat), and returns that URL so the UI can display it as the browser-didn’t-open fallback. Follow with browser_login_wait.
