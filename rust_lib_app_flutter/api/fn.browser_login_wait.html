pub async fn browser_login_wait() -> Result<AuthFlowOutcome, HoundError>
Expand description

Await the browser callback for the pending browser_login_begin / browser_steam_begin flow. A token callback is adopted as a session (persist + connect realtime) → SignedIn; a Steam link-token callback parks the token and returns Unknown for the create-or-link screen. Errors on timeout or when browser_login_cancel interrupts it.
