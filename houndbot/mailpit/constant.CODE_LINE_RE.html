Expand description

Anchors on the plain-text body’s code line (services/backend/src/api/auth/magic.rs): "Or enter this code in the app: {code}". The same email’s link also embeds ?code=..., but this prefix is unambiguous and doesn’t require URL parsing. Alphabet excludes 0/O/1/I, so [A-Z2-9]{12} matches the code exactly.
