Expand description

A running microphone source delivering fixed-size 48 kHz mono PCM frames.

§Implementor contract

  • Construction opened the device and negotiated as close to 48 kHz mono i16 as it allows; it MUST fail loudly (typed AudioError) on no-device / permission-denied — never succeed-then-deliver-silence.
  • Self::next_frame blocks until the next crate::types::SAMPLES_PER_FRAME-sample frame is assembled from the device callback, stamped on the capture clock supplied at open. Returns Err when the stream ends or the device is lost.
  • Whatever the device’s native rate/channels, delivered frames are always 48 kHz mono (the backend downmixes/resamples off the RT callback).
