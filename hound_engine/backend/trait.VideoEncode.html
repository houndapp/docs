Expand description

A video encoder: raw frames in, encoded chunks out, with keyframe and bitrate control. One per outbound video track.

§Implementor contract

  • Self::encode may return zero or more chunks for one input frame: look-ahead encoders buffer internally, so early frames can yield nothing and a later frame can flush several. The caller must not assume 1:1.
  • Self::request_keyframe forces the next encoded frame to be a keyframe (IDR). Callers coalesce PLI storms before driving it.
  • Self::set_bitrate retargets rate control live; best-effort.
  • Self::flush drains any buffered frames at stream end.
