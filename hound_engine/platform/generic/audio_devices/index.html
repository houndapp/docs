§Audio device enumeration (cpal hosts)

Walks cpal’s host device list into typed DeviceDescriptors for the crate::api::Engine device inventory. The enumeration touches the host, so it is only smoke-tested; the dedup/default-flag logic (fold_names) is pure and tested without a sound card.

§The device-id caveat (documented, deliberate)

cpal exposes no stable, portable device identifier — only a human name() string (and even that can error). So the name IS the id here: id == name. Two devices with identical names (e.g. two of the same USB interface) are therefore indistinguishable through this API, and a device that renames itself across reconnect gets a fresh id. This is a known cpal limitation, not a bug to paper over; the camera path (AVFoundation) has a real uniqueID and does not share it. Selection-by-id therefore re-resolves the name against the live host list at open time (see super::cpal_io), so a vanished name falls back to the default rather than failing the call.

collect_named 🔒
Fold a cpal device iterator into deduplicated typed descriptors, marking the one whose name matches default_name. A device whose name errors is skipped (we have no id for it). Delegates the dedup/default logic to fold_names so that logic is unit-testable without a sound card.
device_name 🔒
cpal exposes the device name through Display (its name() accessor was removed in 0.18.1 in favour of the structured description()); the human name IS the id here, so read it via to_string() — the documented string path.
enumerate_microphones 🔒
Enumerate input (microphone) devices on the default host, flagging the current default. Best-effort: a host/device that errors is skipped, never fatal — an empty list is a valid (if unusual) answer. Duplicate names are de-duplicated (first wins) because the name is the id and a collision is unaddressable.
enumerate_speakers 🔒
Enumerate output (speaker) devices on the default host, flagging the current default. Same best-effort + dedup contract as enumerate_microphones.
fold_names 🔒
Fold a stream of device names into deduplicated typed descriptors (name == id), flagging the one matching default_name. Pure (no host access) so the dedup + default logic is tested directly. A duplicate name is unaddressable through the name-is-id contract, so the first occurrence wins.
