pub(crate) fn select_capture_format(
    offered: &[[u8; 4]],
) -> Option<CaptureFormat>
Expand description

Pick the best capture format from the device’s offered FourCCs, applying the preference order (NV12 > YUYV > MJPEG) and ignoring anything unrecognised.

Returns None if the camera offers nothing we can convert (the caller then surfaces a typed UnsupportedFormat). When the only convertible option is MJPEG, it IS returned (so the caller can report exactly that and treat it as the documented JPEG-decode follow-up) — the caller decides whether to refuse; this function’s job is purely the ranking.
