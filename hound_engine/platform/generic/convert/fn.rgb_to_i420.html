pub(crate) fn rgb_to_i420(rgb: &[u8], width: usize, height: usize) -> Vec<u8> ⓘ
Expand description

Convert packed RGB (width*height*3 bytes, R,G,B) to planar I420.

Returns Y (w*h) || U (w/2 * h/2) || V (w/2 * h/2). width and height must be even (I420 chroma is half-resolution); odd inputs are floored, which is safe because the capturer only requests even resolutions.

BT.601 limited range:

  • Y = 0.257R + 0.504G + 0.098B + 16
  • U = -0.148R - 0.291G + 0.439B + 128
  • V = 0.439R - 0.368G - 0.071B + 128
