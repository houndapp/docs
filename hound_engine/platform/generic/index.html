§The portable backend pieces

Everything in here builds and runs on every desktop target (and most of it on mobile too): cpal device I/O, the mono resampler at the device boundary, audio device enumeration, the cross-platform SW H.264 codec (openh264), the synthetic (camera-free) test source, and pixel-format conversion. A new platform gets all of this for free; only the camera + HW codec backends are per-platform.

The rav1e SW AV1 encoder ([rav1e_encoder]) lives here too but is test-only (#[cfg(test)], rav1e is a dev-dependency): it is the AV1 source for the VideoToolbox AV1 HW-decode tests, not a runtime codec. SW fallback is H.264-only — a device with no HW video codec cannot run SW AV1 in real time anyway.
