Convert one raw V4L2 buffer (in the negotiated format) to a tightly-packed I420 buffer at the native geometry. A free function over snapshotted config (not a &self method) so it does not borrow the capturer while the mmap buffer slice from stream.next() is still alive (that slice mutably borrows self.stream, so any concurrent &self borrow would not type-check). None (a skippable frame) on a malformed/short buffer rather than a panic.
