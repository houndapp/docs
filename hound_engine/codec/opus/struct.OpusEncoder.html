Expand description

Opus encoder for the endpoint (48 kHz mono VOIP, 20 ms frames).

Fields§

§enc: Encoder§scratch: Vec<u8>

Reused encode scratch so the per-frame path does not allocate the output buffer each call (only the final exact-length packet Vec is allocated).

impl OpusEncoder

Source

pub fn new(bitrate: Bitrate, profile: OpusProfile) -> Result<Self, AudioError>

Build an encoder at bitrate, tuned by OpusProfile (speech vs music), with FEC enabled. Use OpusProfile::Voice for the mic and OpusProfile::Music for shared screen/system audio.

Source

pub fn encode(&mut self, frame: &PcmFrame) -> Result<OpusPacket, AudioError>

Encode one PCM frame to one Opus packet (1:1 — no look-ahead reorder like the video encoders).

Source

pub fn set_bitrate(&mut self, bitrate: Bitrate) -> Result<(), AudioError>

Retarget the Opus bitrate live. Best-effort, cheap and lossless mid-stream for Opus (a BWE loop can drive this).
