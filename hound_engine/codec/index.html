§Portable codecs

Codec logic that runs identically on every platform (pure Rust / vendored C, no platform API):

  • opus — the voice codec (OpusEncoder/OpusDecoder, libopus via audiopus). There is no per-platform audio codec: Opus is Opus everywhere, so these are concrete types, not backend trait impls.
  • av1_obu — AV1 OBU parsing + av1C configuration-record synthesis. Pure logic, testable on any host; its consumer (the VideoToolbox AV1 decode bootstrap) is platform-gated, the parsing is not.
