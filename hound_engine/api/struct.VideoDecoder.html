Struct VideoDecoder 

Source 
pub struct VideoDecoder(SwVideoDecoder);
Expand description

A video decoder for one inbound track, by wire codec, hiding whether the hardware (VideoToolbox) or the software (openh264) path was selected. Obtained from Engine::video_decoder; dropping it tears the decode session down. Not Send: lives on the decode thread.

A thin newtype over the platform-selected crate::platform::VideoDecoderBackend (VideoToolbox on Apple, SW openh264 elsewhere); both methods delegate, so per-AU decode is static dispatch and this facade carries no platform cfg.

Decode one encoded access unit. Returns the decoded I420 frame if one is ready, or None while pre-bootstrap (no keyframe/sequence header seen yet) or buffering. Errors only on a genuine decode failure.
