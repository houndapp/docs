Struct VideoEncoder

pub struct VideoEncoder(SwVideoEncoder);
Expand description

A real-time video encoder for a negotiated codec, hiding whether the hardware (VideoToolbox: H.264 / HEVC / AV1) or the software (openh264, H.264 only) path was selected. Obtained from Engine::video_encoder (or the H.264-specific Engine::h264_encoder); query VideoEncode::codec for the wire codec. Not Send: construct, drive, and drop on the one encode thread.

A thin newtype over the platform-selected crate::platform::VideoEncoderBackend (a static-dispatch enum on Apple, the SW encoder elsewhere); every method delegates, so there is no dyn on the per-frame hot path and no platform cfg in this facade.
