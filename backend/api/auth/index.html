Email magic-link login: a one-shot code emailed to the address, redeemed at /auth/magic/verify for either a session (known email) or a stateless registration token (unknown email — mirrors the Steam link-token flow in super::steam).

Steam login: Steamworks session-ticket validation (launched through Steam) and OpenID 2.0 assertion verification (browser flow), both resolving to a SteamID64 that maps onto an identity row.
