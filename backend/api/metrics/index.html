Module metrics

Prometheus metrics: the process-wide recorder + render handle, and the HTTP request-instrumentation middleware. /internal/metrics’s handler lives in super::internal (co-located with the other internal-auth-gated routes); this module owns everything that feeds it.

Functions§

handle 🔒
Installs the global recorder and returns its render handle. OnceLock makes repeated calls within one process return the same handle instead of erroring on a second install_recorder() — irrelevant in production (Api::serve calls this once) but keeps the test harness, which constructs a fresh super::Api per test function, safe regardless of how the test runner schedules those calls.
track_http_requests 🔒
Records http_requests_total{method,route,status} and http_request_duration_seconds{method,route,status} for every request.
