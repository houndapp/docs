AuthenticatedClient
A client that has been verified to authentication tokens, and contains all methods accessing the app backend that require authenticated access.
Client
The client for the API backend in the hound application. This type provides general utilities along with methods to access the backend. Note that this type only provides methods which are valid without authentication, to access authenicated methods you need to use AuthenticatedClient through Self::auth or Self::login.
