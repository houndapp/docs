§Room state machine

This module tracks who is in a voice channel and manages how media frames are forwarded from one participant to all others.

The key design principle here is sans-IO: the room state machine never touches the network itself. Instead, callers hand it a state::FrameSink implementation for each peer. In production this is a thin wrapper around a QUIC connection; in tests it’s a [state::MockSink] that collects frames in memory so we can inspect them without any network setup.

This separation makes every piece of forwarding logic unit-testable without standing up real iroh connections.
