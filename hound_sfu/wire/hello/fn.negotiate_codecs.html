pub fn negotiate_codecs(client: &ClientCodecs) -> Option<NegotiatedCodecs>
Expand description

Negotiate codecs from the client’s preferences.

Picks the first entry from each of the client’s preference lists. Returns None if any list is empty, which maps to HelloNack { reason: HelloNackReason::CodecMismatch }.

This is a pure function: no locks, no I/O.

§Example

let prefs = ClientCodecs::default_preferences();
let negotiated = negotiate_codecs(&prefs).unwrap();
assert_eq!(negotiated.cam_video, VideoCodecId::Av1);
assert_eq!(negotiated.audio, AudioCodecId::Opus);
