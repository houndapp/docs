Expand description

Decode a length-prefixed JSON message.

Reads a 4-byte LE u32 length prefix, then deserialises that many bytes of JSON into T. Trailing bytes after the declared length are ignored.

§Errors

Returns Err if the slice is shorter than the declared length, or if the JSON fails to deserialise into T.
