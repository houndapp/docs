pub fn attach(
    stream_key: String,
    native_buffer_ptr: usize,
    texture_id: i64,
    width: u32,
    height: u32,
) -> Result<(), String>
Expand description

Bind stream_key to the native buffer the Flutter plugin registered as texture_id, and start rendering pushed frames into it. Idempotent: an attach under an existing key replaces the previous binding.

native_buffer_ptr is platform-defined: an IOSurface-backed CVPixelBufferRef on macOS (geometry is read off the buffer; width/ height are advisory), a DXGI shared handle to the plugin’s BGRA texture on Windows, and a raw width×height×4 RGBA byte buffer on Linux (neither can be introspected, so the size is load-bearing). It must stay valid until detach returns for this key. Returns an error string (FRB maps it to a Dart exception) on a null pointer or setup failure.
