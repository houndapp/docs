Expand description

Bind stream_key to the plugin’s w×h RGBA buffer at buffer_ptr and start converting the latest pushed frame into it. Replaces any existing stream under the same key (idempotent re-attach — used both for a fresh tile and to rebind after a decoded-resolution change recreates the buffer at a new size). Returns an error string (mapped to a Dart exception by the FRB layer) on a null pointer.

§Safety contract

buffer_ptr must point to at least w*h*4 writable bytes that remain valid until detach returns for this key. The Dart side guarantees this by detaching (stop + join) before destroying the texture/buffer, and by passing the SAME w/h it allocated the buffer with.
